hrengin
=======

## What is this?  ##

This is the core of hrengin (formerly hrEngine), a general purpose game engine.
hrengine is only a library, and can not be used alone.

### Status ###

Current version: hrengin A (1.0/1000)


## Dependencies ##

* Bullet physics
* Irrlicht
* OpenAL Soft
* RakNet
